# Planning

## Versioning
0.1.x is the foundation and early game, already shipped. 0.2 is planets, orbit,
orbital science, and the late game content. Do not advertise 0.2 features as
shipped.

## Milestones
1. M0 Verify the foundation in Factorio. This is blocking. Only JJ can launch
   the game.
2. M1 Real ascent to orbit as a tiled space surface.
3. M2 Orbital science tier, original, made in orbit.
4. M3 Travel to planets by an original mechanic. Colonies, not piloted ships.
5. M4 Planet paradigms. Each planet forces a different way to build. This is the
   core differentiator.
6. M5 Content and polish. Real art, balance, locale.

## Active lanes
The main session takes planet terrain per class and new tiles. The manager team
takes the blueprint break restructure and the Core Tap mechanic, design first,
until M0 passes.
